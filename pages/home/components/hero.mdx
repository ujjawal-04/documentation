# Hero

A responsive React component that creates a visually appealing hero section with gradient backgrounds, feature lists, and responsive product images.

## Syntax

```jsx copy
<Hero />
```

## Description

The Hero component creates a modern, responsive landing section featuring:
- Gradient background
- Gradient text heading
- Feature list with icons
- Responsive product images
- Call-to-action button
- Mobile and desktop layouts

- Heading: `text-4xl font-semibold tracking-tight md:text-4xl lg:text-6xl`
- Subheading: `font-semibold text-[#697294]`
- Description: `text-base leading-7 text-[#11181C] md:text-lg md:leading-8`
- Feature text: `text-sm font-medium leading-tight lg:text-base`

### Gradient Effects
```css copy
/* Background Gradient */
background: linear-gradient(102.84deg, #E1EFFF 26.12%, #B5D7F8 66.91%, #D3EAFF 93.51%)

.hero-title {
  background-gradient-to-r: from-[#1e2c60] to-[#3c53a8];

```

## Responsive Design

### Breakpoints
- Mobile: Single column layout
- Desktop (lg): Two-column layout
- Extra large (xl): Additional padding for product image

### Key Responsive Features
1. Image switching between mobile/desktop versions
2. Column layout changes
3. Typography scaling
4. Feature list layout adaptation
5. Padding and margin adjustments

## Accessibility

### Image Attributes
- Descriptive alt text
- Async decoding
- Proper aspect ratios
- Color transparency fallback

### Text Content
- Semantic heading structure
- Proper text contrast
- Font scaling for readability

## Browser Support

Requires browsers with support for:
- CSS Grid
- CSS Gradients
- Aspect Ratio
- Modern CSS Variables
- Flexbox
- CSS Transitions

## Performance Considerations

1. Images
   - Different versions for mobile/desktop
   - Proper aspect ratios to prevent layout shift
   - Async decoding for performance

2. SVG Icons
   - Inline SVGs for faster loading
   - Proper viewBox settings
   - Optimized paths

3. Gradients
   - Hardware-accelerated when possible
   - Z-index management for proper layering
