# Payment Test

The `Payment Test` component is a simple React component that displays a warning badge for test payment environments. It uses the Medusa UI Badge component to create a consistent, visually distinct notification.

## Overview

```typescript copy
import { Badge } from "@medusajs/ui"

const PaymentTest = ({ className }: { className?: string }) => {
  return (
    <Badge color="orange" className={className}>
      <span className="font-semibold">Attention:</span> For testing purposes
      only.
    </Badge>
  )
}

export default PaymentTest
```

## Props

### className
- Type: `string`
- Required: No
- Description: Optional CSS class name to be applied to the Badge component for custom styling.

## Usage Examples

### Basic Usage

## Best Practices

1. **Placement**
   - Place near payment-related elements
   - Position where it's immediately visible
   - Don't hide or obscure important form elements

2. **Usage**
   - Use in test/development environments
   - Include in payment method selection areas
   - Display before payment processing

3. **Styling**
   - Maintain the orange warning color
   - Ensure text remains readable
   - Keep adequate spacing around the badge
