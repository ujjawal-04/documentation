# Sign In Prompt

A component that displays a prompt encouraging users to sign in to their account, featuring a heading, descriptive text, and a sign-in button.

## Overview

The `Sign In Prompt` component creates a horizontal banner with text content and a call-to-action button for user authentication.

## Technical Details

### Import

```javascript copy
import { Button, Heading, Text } from "@medusajs/ui"
import LocalizedClientLink from "@modules/common/components/localized-client-link"
```

### Component Structure

```jsx copy
const SignInPrompt = () => {
  return (
    <div className="flex items-center justify-between bg-white">
      <ContentSection />
      <ButtonSection />
    </div>
  )
}
```
