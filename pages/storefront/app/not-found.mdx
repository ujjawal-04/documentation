# Not Found

## Overview

This page provides a user-friendly experience when a user navigates to a non-existent route in the application.

## Imports

```typescript copy
import { ArrowUpRightMini } from "@medusajs/icons"
import { Text } from "@medusajs/ui"
import { Metadata } from "next"
import Link from "next/link"
```

### Key Imports
- `ArrowUpRightMini`: An icon component from Medusa UI
- `Text`: A text component from Medusa UI
- `Metadata`: Next.js type for page metadata
- `Link`: Next.js routing component for client-side navigation

## Metadata Configuration

```typescript copy
export const metadata: Metadata = {
  title: "Not Found",
  description: "Page does not exist",
}
```

The metadata provides:
- A clear title indicating the page is not found
- A description explaining the page's status

## Design Considerations
- Provides clear messaging about the error
- Offers a straightforward way to return to the homepage
- Uses consistent styling with application UI

## Best Practices
- Keep error messages clear and concise
- Provide a direct path back to a valid page
- Use subtle animations to improve user experience
- Maintain consistent styling with the rest of the application

## Potential Improvements
- Add analytics tracking for not found events
