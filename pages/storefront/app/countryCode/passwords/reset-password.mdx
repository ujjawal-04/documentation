# Reset Password

## Overview

The Reset Password page provides users with an interface to set a new password after initiating a password reset request.

## Imports

```typescript copy
import React from 'react'
import ResetPassword from '@modules/account/components/reset-password'
import Nav from '@modules/layout/templates/nav'
import Footer from '@modules/layout/templates/footer'
```

### Key Imports
- `React`: Core React library
- `ResetPassword`: Component for actual password reset form
- `Nav`: Navigation component
- `Footer`: Page footer component

## Component Structure

```typescript copy
function page() {
  return (
    <>
      <Nav/>
      <ResetPassword />
      <Footer/>
    </>

}

export default page
```

### Component Characteristics
- Functional component

- `Nav`: Provides top-level navigation

- `Footer`: Provides bottom-level page information and links
