# Request Reset Password

## Overview

The Password Reset page provides users with an interface to request a password reset for their account.

## Imports

```typescript copy
import React from 'react'
import RequestResetPassword from '@modules/account/components/forgot-password'
import Nav from '@modules/layout/templates/nav'
import Footer from '@modules/layout/templates/footer'
```

### Key Imports
- `React`: Core React library
- `RequestResetPassword`: Component for password reset request form
- `Nav`: Navigation component
- `Footer`: Page footer component

## Component Structure

```typescript copy
const page = () => {
  return (
    <>
      <Nav/>
      <RequestResetPassword />
      <Footer/>

}

export default page
```

### Component Characteristics
- Functional component
- Renders three main sections:

  2. Password Reset Request Form

- Simple, straightforward layout

### Layout Composition
- `Nav`: Provides top-level navigation
- `RequestResetPassword`: Central component for password reset functionality
- `Footer`: Provides bottom-level page information and links
