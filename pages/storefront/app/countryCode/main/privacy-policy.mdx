# Privacy Policy 

## Overview

The Privacy Policy component displays comprehensive information about how the company collects, uses, and protects customer data. The component is structured with clear sections covering various aspects of data privacy and security policies, making it easy for users to understand the company's privacy practices.


- `React`: Core React library for component creation
- `Check`: Icon component from Lucide React library used for visual indicators in lists

The component is organized into 10 primary sections:
1. Information We Collect
2. How We Use Your Information
3. SMS/Text Messaging Service
4. Security and Encryption
5. Third-Party Service Providers
6. Data Breach Notification
7. Customer Support
8. GDPR Compliance Notice

### Subsection Structure
Each main section includes:
