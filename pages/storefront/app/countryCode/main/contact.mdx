# Contact

## Overview

The Contact page provides customers with information on how to get in touch with the company. It includes self-service resources to help customers find answers quickly and provides direct contact methods for more personalized support.

```typescript copy
import { Check } from "lucide-react"
```

### Key Imports
- `React`: Core React library
- `Check`: Check icon component from Lucide React icon library
