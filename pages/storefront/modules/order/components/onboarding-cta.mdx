# OnboardingCta

A React component that provides a call-to-action for completing store setup after successfully creating a test order.

## Syntax

```jsx copy
<OnboardingCta orderId="12345" />
```

## Description

The `OnboardingCta` component displays a confirmation message and a button that allows users to finalize the onboarding process in the admin dashboard.

### Features:
- Displays a success message after creating a test order
- Provides a call-to-action button for completing setup
- Uses Medusa UI components for consistent styling

## Technical Details

### Dependencies

```javascript copy
import { resetOnboardingState } from "@lib/data/onboarding"
import { Button, Container, Text } from "@medusajs/ui"
```

### Props

| Prop     | Type   | Description                          |
|----------|--------|--------------------------------------|
| orderId  | string | The unique identifier of the order. |

### Component Structure

The component receives `orderId` as a prop and renders a success message with a button to complete the onboarding setup.

```javascript copy
const OnboardingCta = ({ orderId }: { orderId: string }) => {
  return (
    <Container className="max-w-4xl h-full bg-ui-bg-subtle w-full">
      <div className="flex flex-col gap-y-4 center p-4 md:items-center">
        <Text className="text-ui-fg-base text-xl">
          Your test order was successfully created! ðŸŽ‰
        </Text>
        <Text className="text-ui-fg-subtle text-small-regular">
          You can now complete setting up your store in the admin.
        </Text>
        <Button
          className="w-fit"
          size="xlarge"
          onClick={() => resetOnboardingState(orderId)}

      </div>
    </Container>
  )
}
```

The component follows Medusa UI's styling conventions:

- **Container:** Background color for subtle UI sections (`bg-ui-bg-subtle`)
