# Help

A React component that provides quick access to support resources with links to contact and returns/exchanges pages.

## Syntax

```jsx copy
<Help />
```

## Description

The Help component creates a simple support navigation section that:
- Displays a "Need help?" heading
- Provides direct links to the contact page
- Includes a link to returns and exchanges information
- Uses consistent styling aligned with the application design system

## Technical Details

### Dependencies

```javascript copy
import { Heading } from "@medusajs/ui"
import LocalizedClientLink from "@modules/common/components/localized-client-link"
import React from "react"
```

### Component Structure

The component is a simple functional component with no props:

```javascript copy
const Help = () => {
  return (
    // Component content
  )
}
```

## Styling

The component uses Tailwind CSS and Medusa UI components for styling with the following key features:

### Container
- Top margin for spacing from preceding content (`mt-6`)

### Heading
- Uses Medusa UI's Heading component for consistent typography
- Semi-bold base text class for proper emphasis (`text-base-semi`)

### Content
- Regular base text styling (`text-base-regular`)
- Vertical margin for spacing (`my-2`)
- Flex column layout for link list (`flex flex-col`)
- Consistent spacing between links (`gap-y-2`)

## Technical Considerations

- Simple component with no props or state management
- Uses Medusa UI's Heading component for consistent typography
- Uses LocalizedClientLink for localization support
- Both links currently point to the same "/contact" URL
- Minimal, focused component with a single responsibility

## Accessibility

- Uses semantic HTML with proper heading and list elements
- Follows a logical content hierarchy
- Links have descriptive text
- Maintains consistent spacing for usability
- Part of a cohesive support interface system

## Browser Support

Requires browsers with support for:
- Modern JavaScript (ES6+)
- CSS Flexbox
