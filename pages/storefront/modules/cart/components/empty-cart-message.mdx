# Empty Cart Message

The `Empty Cart Message` component displays a message when the user's shopping cart is empty, providing a link to browse products.

## Overview

This component creates a user-friendly empty state for the shopping cart, featuring a heading, descriptive text, and a call-to-action link to the store.

## Syntax

```jsx copy
<EmptyCartMessage />
```

## Technical Details

### Import

```jsx copy
import EmptyCartMessage from "@modules/cart/components/empty-cart-message"
```

### Dependencies

```jsx copy
import { Heading, Text } from "@medusajs/ui"
import InteractiveLink from "@modules/common/components/interactive-link"
```

## Component Structure

### Layout
The component uses a flex container with the following structure:
- Main container with vertical padding and centered content
- Heading "Cart"
- Descriptive text
- Interactive link to store

  <Heading>Cart</Heading>

## Examples

### Basic Usage

```jsx copy
function CartPage() {
  const isEmpty = true;
  
  return (
    <div className="cart-container">
      {isEmpty && <EmptyCartMessage />}
    </div>
  )
}
```

### Within a Cart Context

```jsx copy
function Cart() {
  const { items } = useCart();
  
  return (
    <div>
      {items.length === 0 ? (
        <EmptyCartMessage />
      ) : (
        <CartItems items={items} />
      )}
    </div>
  )
}
```
