# Footer

A React component that renders a comprehensive, responsive website footer with newsletter subscription, navigation menus, contact information, and legal disclaimers.

## Syntax

```jsx copy
<Footer />
```

## Description

The Footer component creates a full-featured website footer that:
- Provides a newsletter subscription form
- Organizes navigation links into logical categories (Peptides, Support, Company, Policies)
- Displays company contact information
- Shows certification and payment method images
- Includes necessary legal disclaimers
- Features responsive layouts for both mobile and desktop viewports
- Uses a dark blue color scheme with white and gray text

## Styling

The component uses Tailwind CSS for styling with the following key features:

### Color Scheme
- Dark blue background (`bg-[#002B54]`)
- Lighter blue for newsletter section (`bg-[#183F63]`)
- Teal for action buttons (`bg-teal-600`)
- White for primary text (`text-white`)
- Light gray for secondary text (`text-gray-300`, `text-[#CCD5DD]`)
- Dark borders for separation (`border-[#335576]`)

### Responsive Design
- Mobile-first approach with different layouts for small and large screens
- Conditional display using `block md:hidden` and `hidden md:grid`
- Flexible widths and layouts with grid and flex containers
- Responsive text sizing and spacing
- Different column structures based on viewport size

### Typography
- Varied text sizes for hierarchy (heading: `text-xl`, body: `text-sm`, disclaimer: `text-xs`)
- Semibold font for section headers
- Proper spacing with margin utilities
- Hover states for links (`hover:text-white`)

### Layout
- Max-width container for consistent alignment (`max-w-[1440px]`)
- Grid system for structured layouts
- Flexbox for alignment and spacing
- Generous spacing between sections

## Technical Considerations

- Pure presentational component with no props or state management
- Uses Next.js Image component for optimized image loading
- Contains several hardcoded links and content that may need updating
- Newsletter form does not include submission handling
- Uses Lucide React for the mail icon
- Includes detailed responsive breakpoints for different screen sizes
- Contains important legal and contact information

## Accessibility

- Uses semantic HTML with appropriate heading levels
- Maintains adequate color contrast for readability
- Provides alt text for images
- Organizes navigation into logical groups
- Uses hover states for interactive elements
- Maintains appropriate text sizes for readability

## Browser Support

Requires browsers with support for:
- Modern JavaScript (ES6+)
- CSS Grid and Flexbox
- CSS Variables (custom properties)
- Media queries for responsive design
- Modern image formats through Next.js Image optimization
