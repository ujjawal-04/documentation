# Medusa CTA

A React component that displays a horizontal list of legal policy links, typically used in a footer or important information section.

## Syntax

```jsx copy
<MedusaCTA />
```

## Description

The Medusa CTA component creates a simple, responsive layout for essential policy links. It:
- Displays three important legal links (Shipping & Returns, Terms & Conditions, Privacy Policy)
- Uses a consistent, accessible text style
- Adjusts text size and spacing based on screen size
- Maintains equal spacing between links
- Centers the links horizontally

## Technical Details

### Dependencies

```javascript copy
import { Text } from "@medusajs/ui"
```

### Component Structure

The component is a simple functional component with no props:

```javascript copy
const MedusaCTA = () => {
  return (
    // Component content
  )
}
```

## Layout Structure

### Container
```jsx copy
<Text className="flex w-full flex-row items-center justify-center gap-2 py-2 md:gap-4">
  {/* Link elements */}
</Text>
```

### Link Elements
```jsx copy
<a href="/shipping-and-returns" rel="noreferrer" className="text-sm text-gray-600 md:text-base">
  Shipping & Returns
</a>
<a href="/terms-and-conditions" rel="noreferrer" className="text-sm text-gray-600 md:text-base">
  Terms & Conditions
</a>
<a href="/privacy-policy" rel="noreferrer" className="text-sm text-gray-600 md:text-base">
  Privacy Policy
</a>
```

## Styling

The component uses Tailwind CSS for styling with the following key features:

### Responsive Design
- Smaller text on mobile devices (`text-sm`)
- Larger text on medium screens and up (`md:text-base`)
- Smaller gap between links on mobile (`gap-2`)
- Larger gap between links on medium screens and up (`md:gap-4`)

### Layout
- Flex container with row direction (`flex flex-row`)
- Full width container (`w-full`)
- Centered content horizontally (`justify-center`)
- Vertically aligned items (`items-center`)
- Consistent vertical padding (`py-2`)

### Typography
- Muted text color for links (`text-gray-600`)
- Consistent font size based on screen size
- Uses the Text component from Medusa UI for typography consistency

## Technical Considerations

- Simple component with no props or state management
- Uses Medusa UI's Text component for consistent typography
- Links use `rel="noreferrer"` for security and privacy
- Hard-coded URLs assume specific routes exist in the application
- Links open in the same window/tab (no `target="_blank"`)

## Accessibility

- Uses semantic HTML with proper anchor (`<a>`) elements
- Maintains sufficient color contrast for readability
- Provides adequate spacing between clickable elements
- Scales appropriately with text size changes
- Consistent with platform navigation patterns

## Browser Support

Requires browsers with support for:
- Modern JavaScript (ES6+)
- CSS Flexbox
- Media queries for responsive design
