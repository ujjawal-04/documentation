# Medusa Icon

The `Medusa` component is an SVG icon that displays the Medusa Commerce logo. It's primarily used to represent the Medusa e-commerce platform in navigation, footers, or branding elements within applications built with Medusa.

## Features

- Customizable size and color
- Detailed Medusa logo representation
- Default color optimized for UI contexts
- Responsive scaling
- Support for additional SVG attributes

## Props

| Prop | Type | Required | Description |
|------|------|----------|-------------|
| `size` | `string` | No | Base size value for the icon (default: "20") |
| `color` | `string` | No | Color of the icon fill (default: "#9CA3AF") |
| `...attributes` | `SVGAttributes<SVGElement>` | No | Additional attributes passed to the underlying SVG element |

## Usage

```tsx copy
import Medusa from "@components/icons/Medusa"

export default function Footer() {
  return (
    <footer className="py-6 border-t">
      <div className="container mx-auto">
        <div className="flex items-center gap-2">
          <Medusa size="24" />
          <span className="text-ui-fg-subtle">Powered by Medusa Commerce</span>
        </div>
      </div>
    </footer>
  )
}
```

## Styling

The icon is designed with:

- Clean SVG paths for accurate logo representation
- Fixed aspect ratio for brand integrity
- Default color set to a subtle gray (#9CA3AF)
- Proper viewBox settings for consistent rendering
- Single path for optimal rendering

## Customization

### Size Variations

```tsx copy
// Small
<Medusa size="16" />

// Default
<Medusa size="20" />

// Medium
<Medusa size="24" />

// Large
<Medusa size="32" />
```

### Color Variations

```tsx copy

// Brand color
<Medusa color="#7C3AED" />

// White (for dark backgrounds)
<Medusa color="#FFFFFF" />
```

## Accessibility

When using this icon in interfaces:

- When used as a standalone interactive element, provide appropriate `aria-label` attributes
- Ensure sufficient color contrast, especially when using non-default colors
- For decorative usage, consider adding `aria-hidden="true"` to the SVG
- When used in branding contexts, ensure proper text alternatives are available for screen readers

The component uses:

- React's functional component pattern with TypeScript support
