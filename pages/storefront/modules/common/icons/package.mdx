# Package Icon

The `Package` component is an SVG icon representing a 3D cube or package. It's commonly used to indicate physical products, shipping packages, parcels, or deliverable items in e-commerce and logistics interfaces.

## Features

- Customizable size and color
- Consistent stroke styling
- Clear 3D package/box visual representation
- Responsive scaling
- Support for additional SVG attributes

## Props

| Prop | Type | Required | Description |
|------|------|----------|-------------|
| `size` | `string` | No | Width and height of the icon in pixels (default: "20") |
| `color` | `string` | No | Color of the icon's stroke (default: "currentColor") |
| `...attributes` | `SVGAttributes<SVGElement>` | No | Additional attributes passed to the underlying SVG element |

## Usage

```tsx copy
import Package from "@components/icons/Package"

export default function OrderSummary() {
  return (
    <div className="shipping-info">
      <div className="flex items-center gap-2 mb-2">
        <Package size="20" />
        <h3 className="font-medium">Your Order</h3>
      </div>
      <p className="text-ui-fg-subtle">2 items, estimated delivery Jun 15-18</p>
    </div>
  )
}
```

## Styling

The icon is designed with:

- Clean paths for a clear 3D box representation
- Consistent stroke width (1.5) for visual harmony with other icons
- Rounded stroke caps and joins for a polished appearance
- Default color inheritance via "currentColor"
- Proper viewBox settings for responsive scaling

## Customization

### Size Variations

```tsx copy
// Small
<Package size="16" />

// Default
<Package size="20" />

// Medium
<Package size="24" />

// Large
<Package size="32" />
```

### Color Variations

```tsx copy
// Default (inherits from text color)
<Package />

// Specific color
<Package color="#3B82F6" />

// UI token colors
<Package color="var(--color-ui-fg-interactive)" />
```

## Accessibility

When using this icon in interfaces:

- When used in interactive elements, provide appropriate `aria-label` attributes
- Ensure sufficient color contrast for visibility
- Use the icon consistently throughout your interface for product/package-related elements
- Consider adding descriptive text alongside the icon for improved clarity
- For icon-only buttons, ensure proper focus states and labels

## Technical Details

The component uses:

- React's functional component pattern with TypeScript support
- SVG paths with detailed drawing commands for a 3D package representation
- Consistent stroke properties across all paths
- Default prop values for ease of use
- Proper typing via the imported `IconProps` interface
