# PayPal Icon

The `PayPal` component is an SVG icon that displays the recognizable PayPal payment symbol with a dual-path design. It's commonly used in checkout flows, payment method selectors, and other UI elements related to PayPal payment options.

## Features

- Customizable size and appearance
- Consistent stroke width and styling
- Responsive scaling with proper viewBox
- Support for additional SVG attributes
- Clean, recognizable PayPal symbol with dual-path styling

## Props

| Prop | Type | Required | Description |
|------|------|----------|-------------|
| `height` | `string` | No | Height of the icon in pixels (default: "20") |
| `width` | `string` | No | Width of the icon in pixels (default: "20") |
| `...attributes` | `SVGAttributes<SVGElement>` | No | Additional attributes passed to the underlying SVG element |

## Usage

```tsx copy
import PayPal from "@components/icons/PayPal"

export default function PaymentOptions() {
  return (
    <div className="payment-methods flex gap-4 py-4">
      <button 
        onClick={() => selectPaymentMethod('paypal')} 
        className="flex items-center gap-2 p-3 border rounded hover:border-blue-500"
      >
        <PayPal />
        <span>Pay with PayPal</span>
      </button>
    </div>
  )
}
```

## Styling

The icon is designed with:

- Clean SVG paths for crisp rendering at any size
- Consistent stroke width (2) for all paths
- Rounded stroke caps and joins for a polished appearance
- Default color inheritance via configurable stroke color
- Proper viewBox settings (0 0 26 25) for responsive scaling

## Customization

### Size Variations

```tsx copy
// Default
<PayPal />

// Medium
<PayPal width="30" height="30" />

// Large
<PayPal width="40" height="40" />

// Custom with preserved aspect ratio
<PayPal width="50" height="48" />
```

### Color Variations

```tsx copy
// Default color
<PayPal />

// With custom stroke via style override
<PayPal style={{ stroke: "#0070ba" }} />

// Contextual colors
<style jsx>{`
  .paypal-icon path {
    stroke: var(--color-brand);
  }
`}</style>
<PayPal className="paypal-icon" />
```

### With Additional SVG Attributes

```tsx copy
<PayPal 
  strokeWidth="1.5"
  className="payment-icon"
  style={{ filter: "drop-shadow(0px 1px 2px rgba(0, 0, 0, 0.15))" }}
/>
```

## Accessibility

When using this icon in interactive elements:

- Always provide appropriate `aria-label` attributes for icon-only buttons
- Consider adding descriptive text alongside the icon when used in payment contexts
- Ensure sufficient color contrast between the icon and its background
- Include proper focus states for keyboard navigation
- Use semantic HTML elements for interactive payment components

## Technical Details

The component uses:

- React's functional component pattern
- SVG paths with precise dimensions for accurate rendering
- Fixed viewBox settings (0 0 26 25) to maintain proportions
- Dual-path design for the distinctive PayPal logo representation
- Standard stroke attributes for consistent styling
- Rounded caps and joins for a polished appearance
