# Back Icon

The `Back` component is an SVG icon that displays a back arrow combined with a circular return motion. It's commonly used for navigation elements, "go back" buttons, and other UI elements indicating a return to a previous page or state.

## Features

- Customizable size and color
- Consistent stroke width and styling
- Responsive scaling
- Support for additional SVG attributes
- Clean, recognizable "back" symbol with circular return indicator

## Props

| Prop | Type | Required | Description |
|------|------|----------|-------------|
| `size` | `string` | No | Width and height of the icon in pixels (default: "16") |
| `color` | `string` | No | Color of the icon's stroke (default: "currentColor") |
| `...attributes` | `SVGAttributes<SVGElement>` | No | Additional attributes passed to the underlying SVG element |

## Usage

```tsx copy
import Back from "@components/icons/Back"

export default function NavigationHeader() {
  return (
    <div className="header flex items-center py-4">
      <button 
        onClick={() => history.goBack()} 
        className="flex items-center gap-2 text-ui-fg-subtle hover:text-ui-fg-base"
      >
        <Back size="20" />
        <span>Back to products</span>
      </button>
    </div>
  )
}
```

## Styling

The icon is designed with:

- Clean SVG paths for crisp rendering at any size
- Consistent stroke width (1.5) for all paths
- Rounded stroke caps and joins for a polished appearance
- Default color inheritance via "currentColor"
- Proper viewBox settings for responsive scaling

## Customization

### Size Variations

```tsx copy
// Small (default)
<Back size="16" />

// Medium
<Back size="24" />

// Large
<Back size="32" />

// Custom
<Back size="40" />
```

### Color Variations

```tsx copy
// Default (inherits from text color)
<Back />

// Brand color
<Back color="#4F46E5" />

// Contextual colors
<Back color="var(--color-error)" />
<Back color="var(--color-success)" />

// Semantic UI colors
<Back color="var(--color-ui-fg-interactive)" />
```

### With Additional SVG Attributes

```tsx copy
<Back 
  strokeWidth="2"
  className="rotate-45"
  style={{ filter: "drop-shadow(0px 1px 2px rgba(0, 0, 0, 0.25))" }}
/>
```

## Examples

### Basic Navigation Button

```tsx copy
<button 
  onClick={() => router.back()} 
  className="inline-flex items-center gap-1 text-sm text-ui-fg-subtle hover:text-ui-fg-base"
>
  <Back size="14" />
  <span>Back</span>
</button>
```

### Standalone Icon Button

```tsx copy
<button 
  aria-label="Go back" 
  onClick={() => navigate(-1)}
  className="flex items-center justify-center w-8 h-8 rounded-full bg-ui-bg-base hover:bg-ui-bg-base-hover"
>
  <Back size="20" />
</button>
```

### Within a Header

```tsx copy
<header className="border-b border-ui-border-base px-4 py-3">
  <div className="flex items-center">
    <button 
      onClick={() => history.goBack()} 
      className="mr-4"
      aria-label="Go back"
    >
      <Back size="20" />
    </button>
    <h1 className="text-lg font-medium">Product Details</h1>
  </div>
</header>
```

### Custom Animation

```tsx copy
<style jsx>{`
  .back-icon {
    transition: transform 0.2s ease;
  }
  
  .back-icon:hover {
    transform: translateX(-4px);
  }
`}</style>

<Back size="24" className="back-icon" />
```

## Accessibility

When using this icon in interactive elements:

- Always provide appropriate `aria-label` attributes for icon-only buttons
- Consider adding appropriate focus styles to enhance keyboard navigation
- Ensure sufficient color contrast between the icon and its background
- Pair with text when used for important navigation to improve clarity

## Technical Details

The component uses:

- React's functional component pattern with TypeScript support
- SVG paths for drawing the icon geometry
- prop spreading for flexible attribute handling
- Default prop values for consistent appearance
- Proper typing via the imported `IconProps` interface
