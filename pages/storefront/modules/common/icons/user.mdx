# User Icon

The `User` component is an SVG icon that displays a recognizable user or profile symbol. It's commonly used for user avatars, profile buttons, account sections, and other UI elements that represent a person or user account.

## Features

- Customizable size and color
- Consistent stroke width and styling
- Responsive scaling
- Support for additional SVG attributes
- Clean, recognizable user profile design with head and shoulders silhouette

## Props

| Prop | Type | Required | Description |
|------|------|----------|-------------|
| `size` | `string` | No | Width and height of the icon in pixels (default: "16") |
| `color` | `string` | No | Color of the icon's stroke (default: "currentColor") |
| `...attributes` | `IconProps & SVGAttributes<SVGElement>` | No | Additional attributes passed to the underlying SVG element |

## Usage

```tsx copy
import User from "@components/icons/User"

export default function ProfileButton() {
  return (
    <button 
      onClick={() => openUserMenu()} 
      className="flex items-center gap-2 p-2 rounded-md hover:bg-ui-bg-base-hover"
    >
      <User size="18" />
      <span>My Profile</span>
    </button>
  )
}
```

## Styling

The icon is designed with:

- Clean SVG paths for crisp rendering at any size
- Consistent stroke width (1.5) for all paths
- Rounded stroke caps and joins for a polished appearance
- Default color inheritance via "currentColor"
- Proper viewBox settings (0 0 20 20) for responsive scaling
- Two distinct paths that form a complete user silhouette:
  1. The upper circular head/face area
  2. The lower shoulders/body area with a slight indentation for the neck

## Customization

### Size Variations

```tsx copy
// Small (default)
<User size="16" />

// Medium
<User size="24" />

// Large
<User size="32" />

// Custom
<User size="40" />
```

### Color Variations

```tsx copy
// Default (inherits from text color)
<User />

// Brand color
<User color="#4F46E5" />

// Muted variant
<User color="var(--color-ui-fg-subtle)" />

// High contrast
<User color="var(--color-ui-fg-base)" />
```

### With Additional SVG Attributes

```tsx copy
<User 
  size="24"
  strokeWidth="2"
  className="my-user-icon"
  style={{ filter: "drop-shadow(0px 1px 2px rgba(0, 0, 0, 0.1))" }}
/>
```

## Accessibility

When using this icon for user-related elements:

- Always provide appropriate `aria-label` attributes for icon-only buttons
- Consider adding tooltips for icon-only buttons to enhance discoverability
- Ensure sufficient color contrast between the icon and its background
- Use semantic HTML elements for interactive elements (buttons, links)
- When used as a placeholder for missing user avatars, consider adding an appropriate `alt` attribute
- For navigation items, consider using `aria-current="page"` for the active page

## Technical Details

The component uses:

- React's functional component pattern with TypeScript support
- SVG paths for drawing the user silhouette geometry
- Type import from a shared `IconProps` interface
- Default prop values for consistent appearance
- Two distinct paths that form the user profile representation:
  1. A circular path for the head/face area
  2. A curved path for the shoulders/upper body
