# Map Pin Icon

The `MapPin` component is an SVG icon that represents a location pin marker. It's commonly used to indicate geographical locations, store locations, delivery addresses, or as map markers in location-based interfaces.

## Features

- Customizable size and color
- Consistent stroke styling
- Clear location marker visual representation
- Responsive scaling
- Support for additional SVG attributes

## Props

| Prop | Type | Required | Description |
|------|------|----------|-------------|
| `size` | `string` | No | Width and height of the icon in pixels (default: "20") |
| `color` | `string` | No | Color of the icon's stroke (default: "currentColor") |
| `...attributes` | `SVGAttributes<SVGElement>` | No | Additional attributes passed to the underlying SVG element |

## Usage

```tsx copy
import MapPin from "@components/icons/MapPin"

export default function StoreLocator() {
  return (
    <div className="store-location">
      <div className="flex items-center gap-2 mb-2">
        <MapPin size="20" />
        <h3 className="font-medium">Store Location</h3>
      </div>
      <p className="text-ui-fg-subtle">123 Main Street, City, Country</p>
    </div>
  )
}
```

## Styling

The icon is designed with:

- Clean paths for clear visual representation
- Consistent stroke width (1.5) for visual harmony with other icons
- Rounded stroke caps and joins for a polished appearance
- Default color inheritance via "currentColor"
- Proper viewBox settings for responsive scaling

## Customization

### Size Variations

```tsx copy
// Small
<MapPin size="16" />

// Default
<MapPin size="20" />

// Medium
<MapPin size="24" />

// Large
<MapPin size="32" />
```

### Color Variations

```tsx copy
// Default (inherits from text color)
<MapPin />

// Specific color
<MapPin color="#EF4444" />

// UI token colors
<MapPin color="var(--color-ui-fg-interactive)" />
```

## Accessibility

When using this icon in interfaces:

- Provide appropriate `aria-label` attributes for any buttons or interactive elements using this icon
- Ensure sufficient color contrast for visibility
- Use the icon consistently throughout your interface for location-related elements
- Consider adding descriptive text alongside the icon for improved clarity
- For interactive map pins, ensure they can be accessed via keyboard navigation
