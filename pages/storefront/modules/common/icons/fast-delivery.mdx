# Fast Delivery Icon

The `FastDelivery` component is an SVG icon that depicts a fast delivery service or shipping option. It's commonly used in e-commerce interfaces, order tracking sections, and shipping method selection interfaces to represent expedited delivery options.

## Features

- Customizable size and color
- Consistent stroke styling
- Detailed delivery vehicle representation
- Responsive scaling
- Support for additional SVG attributes

## Props

| Prop | Type | Required | Description |
|------|------|----------|-------------|
| `size` | `string` | No | Width and height of the icon in pixels (default: "16") |
| `color` | `string` | No | Color of the icon's stroke (default: "currentColor") |
| `...attributes` | `SVGAttributes<SVGElement>` | No | Additional attributes passed to the underlying SVG element |

## Usage

```tsx copy
import FastDelivery from "@components/icons/FastDelivery"

export default function ShippingOptions() {
  return (
    <div className="shipping-options">
      <div className="option flex items-center gap-3 p-3 border rounded mb-2">
        <FastDelivery size="24" />
        <div>
          <h3 className="font-medium">Express Delivery</h3>
          <p className="text-sm text-ui-fg-subtle">Delivery within 1-2 business days</p>
        </div>
      </div>
    </div>
  )
}
```

## Styling

The icon is designed with:

- Detailed paths for accurate representation of the delivery concept
- Consistent stroke width (1.5) for visual harmony with other icons
- Rounded stroke caps and joins for a polished appearance
- Default color inheritance via "currentColor"
- Proper viewBox settings for responsive scaling

## Customization

### Size Variations

```tsx copy
// Small
<FastDelivery size="16" />

// Medium
<FastDelivery size="24" />

// Large
<FastDelivery size="32" />

// Custom
<FastDelivery size="48" />
```

### Color Variations

```tsx copy
// Default (inherits from text color)
<FastDelivery />

// Specific color
<FastDelivery color="#4F46E5" />

// UI token colors
<FastDelivery color="var(--color-ui-fg-interactive)" />
```

## Accessibility

When using this icon in interfaces:

- When used without accompanying text, provide appropriate `aria-label` attributes
- Ensure sufficient color contrast for visibility
- For interactive elements, maintain proper focus states
- Consider adding text labels alongside the icon for clarity
- Use the icon consistently throughout your interface for shipping-related content

## Technical Details

The component uses:

- React's functional component pattern with TypeScript support
- Detailed SVG paths for accurate representation
- Consistent stroke properties across all paths
- Default prop values for ease of use
- Proper typing via the imported `IconProps` interface
