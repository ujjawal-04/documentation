# NextJs Icon

The `NextJs` component is an SVG icon that displays the Next.js framework logo. It's commonly used to represent the Next.js framework in technology stack indicators, documentation, or integration contexts within applications.

## Features

- Customizable size and color
- Detailed Next.js logo representation
- Default color optimized for UI contexts
- Responsive scaling
- Support for additional SVG attributes

## Props

| Prop | Type | Required | Description |
|------|------|----------|-------------|
| `size` | `string` | No | Base size value for the icon (default: "20") |
| `color` | `string` | No | Color of the icon fill (default: "#9CA3AF") |
| `...attributes` | `SVGAttributes<SVGElement>` | No | Additional attributes passed to the underlying SVG element |

## Usage

```tsx copy
import NextJs from "@components/icons/NextJs"

export default function TechStack() {
  return (
    <div className="tech-stack">
      <div className="flex items-center gap-2">
        <NextJs size="24" />
        <span>Built with Next.js</span>
      </div>
    </div>
  )
}
```

## Styling

The icon is designed with:

- Precise SVG path for accurate Next.js logo representation
- Fixed aspect ratio for brand integrity
- Default color set to a subtle gray (#9CA3AF)
- Proper viewBox settings for consistent rendering
- Single detailed path for optimal rendering

## Customization

### Size Variations

```tsx copy
// Small
<NextJs size="16" />

// Default
<NextJs size="20" />

// Medium
<NextJs size="24" />

// Large
<NextJs size="32" />
```

### Color Variations

```tsx copy
// Default (subtle gray)
<NextJs />

// Black (official color)
<NextJs color="#000000" />

// White (for dark backgrounds)
<NextJs color="#FFFFFF" />

// Brand color adaptation
<NextJs color="var(--color-ui-fg-interactive)" />
```

## Accessibility

When using this icon in interfaces:

- When used as a standalone interactive element, provide appropriate `aria-label` attributes
- Ensure sufficient color contrast, especially when using non-default colors
- For decorative usage, consider adding `aria-hidden="true"` to the SVG
- When used to represent technology, provide adequate text context nearby
- Consider using the official black color for best brand recognition

## Technical Details

The component uses:

- React's functional component pattern with TypeScript support
- SVG with fixed internal dimensions for optimal rendering
- Single detailed path for accurate logo representation
- Default prop values optimized for common usage scenarios
- Proper typing via the imported `IconProps` interface
