# Chevron Down Icon

The `ChevronDown` component is a simple, clean SVG icon displaying a downward-pointing chevron arrow. It's commonly used to indicate expandable content, dropdown menus, accordion interfaces, or to prompt users to scroll down.

## Features

- Customizable size and color
- Clean, minimal design
- Consistent stroke styling
- Responsive scaling
- Support for additional SVG attributes

## Props

| Prop | Type | Required | Description |
|------|------|----------|-------------|
| `size` | `string` | No | Width and height of the icon in pixels (default: "16") |
| `color` | `string` | No | Color of the icon's stroke (default: "currentColor") |
| `...attributes` | `SVGAttributes<SVGElement>` | No | Additional attributes passed to the underlying SVG element |

## Usage

```tsx copy
import ChevronDown from "@components/icons/ChevronDown"

export default function Dropdown() {
  const [isOpen, setIsOpen] = useState(false)
  
  return (
    <div className="relative">
      <button 
        onClick={() => setIsOpen(!isOpen)}
        className="flex items-center justify-between w-full px-3 py-2 border rounded"
      >
        <span>Select an option</span>
        <ChevronDown className={isOpen ? "rotate-180 transform" : ""} />
      </button>
      {isOpen && (
        <div className="absolute w-full mt-1 border rounded bg-white">
          {/* Dropdown content */}
        </div>
      )}
    </div>
  )
}
```

## Styling

The icon is designed with:

- Simple, clear path for optimal recognition
- Consistent stroke width (1.5) for visual harmony with other icons
- Rounded stroke caps and joins for a polished appearance
- Default color inheritance via "currentColor"
- Proper viewBox settings for responsive scaling

## Customization

### Size Variations

```tsx copy
// Small
<ChevronDown size="12" />

// Default
<ChevronDown size="16" />

// Medium
<ChevronDown size="20" />

// Large
<ChevronDown size="24" />
```

### Color Variations

```tsx copy
// Default (inherits from text color)
<ChevronDown />

// Specific color
<ChevronDown color="#4F46E5" />

// UI token colors
<ChevronDown color="var(--color-ui-fg-muted)" />
```

### Rotation for Different Directions

```tsx copy
// Up
<ChevronDown className="rotate-180 transform" />

// Right
<ChevronDown className="rotate-270 transform" />

// Left
<ChevronDown className="rotate-90 transform" />
```

## Accessibility

When using this icon in interfaces:

- Add appropriate `aria-expanded` attributes when using in expandable content
- Ensure sufficient color contrast for visibility
- For icon-only buttons, include proper `aria-label` attributes
- Consider adding text labels alongside the icon for clarity
- Use consistent direction conventions throughout your interface
