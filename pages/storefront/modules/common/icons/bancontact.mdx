# bancontact Icon

The `bancontact` component is an SVG icon that displays the Bancontact logo, commonly used in payment method selection interfaces. This component allows for consistent representation of the Bancontact payment option within your application.

## Features

- Customizable color
- Consistent styling
- Accessible SVG implementation with proper title
- Support for additional SVG attributes
- Fixed aspect ratio for brand recognition

## Props

| Prop | Type | Required | Description |
|------|------|----------|-------------|
| `size` | `string` | No | Base size value for the icon (default: "20") |
| `color` | `string` | No | Color of the icon (default: "currentColor") |
| `...attributes` | `SVGAttributes<SVGElement>` | No | Additional attributes passed to the underlying SVG element |

## Usage

```tsx copy
import bancontact from "@components/icons/bancontact"

export default function PaymentMethods() {
  return (
    <div className="payment-options">
      <div className="flex items-center gap-3 p-3 border rounded mb-2">
        <bancontact />
        <span>Pay with Bancontact</span>
      </div>
    </div>
  )
}
```

## Styling

The icon is designed with:

- Clean SVG paths for accurate representation of the Bancontact logo
- Proper viewBox settings for consistent rendering
- Accessible implementation with role="img" and title
- Maintains fixed width and height of 24px for brand integrity
- Customizable fill color

## Customization

### Color Variations

```tsx copy
// Default (inherits from text color)
<bancontact />

// Brand color
<bancontact color="#005498" />

// Grayscale for disabled state
<bancontact color="#9CA3AF" />
```

### With Additional SVG Attributes

```tsx copy
<bancontact 
  className="payment-icon"
  style={{ filter: "drop-shadow(0px 1px 2px rgba(0, 0, 0, 0.1))" }}
/>
```

## Accessibility

When using this icon in interfaces:

- The SVG includes a proper `role="img"` attribute
- A descriptive `<title>` element is included for screen readers
- When used in interactive elements, ensure proper labels are provided
- Maintain sufficient color contrast for visibility
- Consider adding text labels alongside the icon for clarity
