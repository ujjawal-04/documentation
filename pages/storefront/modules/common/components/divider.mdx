# Divider

The `Divider` component provides a consistent way to create visual separation between different sections of content. It renders as a thin horizontal line with configurable styling.

## Features

- Simple, consistent horizontal line
- Customizable appearance through className prop
- Lightweight implementation
- Consistent default styling

## Props

| Prop | Type | Required | Description |
|------|------|----------|-------------|
| `className` | `string` | No | Additional CSS classes to customize the divider's appearance |

## Usage

```tsx copy
import Divider from '@/components/Divider'

export default function ProductDetails() {
  return (
    <div className="product-container">
      <h2>Product Title</h2>
      <p>Product description goes here...</p>
      
      <Divider />
      
      <div className="price-section">
        <span>$99.99</span>
        <button>Add to Cart</button>
      </div>
    </div>
  )
}
```

## Styling

The component uses Tailwind CSS for styling through the Medusa UI `clx` utility and includes:

- 1px height for subtle separation
- Full width by default
- Light gray border color (border-gray-200)
- Small top margin (mt-1)

## Examples

### Basic Usage

```tsx copy
<Divider />
```

### With Custom Styling

```tsx copy

```

### With Dark Theme

```tsx copy
<Divider className="border-gray-700" />
```

### With Vertical Spacing

```tsx copy

```

### In a Card Component

```tsx copy
<div className="bg-white p-4 rounded shadow">
  <h3>Card Title</h3>
  <Divider className="my-2" />
  <p>Card content goes here...</p>

```

## Dependencies

- @medusajs/ui (for the clx utility)
- react (for component structure)

## Technical Details

The component:

- Is implemented as a simple functional component
- Uses the clx utility from Medusa UI for class name composition
- Renders a single div element with border styling to create the line
- Has minimal overhead due to its simplicity
- Maintains consistent styling across the application
- Can be easily customized with additional classes
