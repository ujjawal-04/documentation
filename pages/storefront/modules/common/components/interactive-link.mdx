# Interactive Link

The `Interactive Link` component provides a consistent, styled link that includes hover effects and localization support. It's designed to be used for interactive elements that navigate to other pages while maintaining a button-like appearance.

## Features

- Consistent button-like styling
- Hover effect for better interactivity
- Localization support through LocalizedClientLink
- Click handling capability
- Responsive design
- Custom styling with Medusa UI Text component

## Props

| Prop | Type | Required | Description |
|------|------|----------|-------------|
| `href` | `string` | Yes | The URL the link should navigate to |
| `children` | `React.ReactNode` | No | The content to display inside the link |
| `onClick` | `function` | No | Function to call when the link is clicked |
| `...props` | `any` | No | Any additional props to pass to the underlying LocalizedClientLink |

## Usage

```tsx copy
import InteractiveLink from '@/components/InteractiveLink'

export default function ProductPage() {
  return (
    <div className="product-actions mt-4">
      <h2>Product Name</h2>
      <p>Product description goes here...</p>
      
      <div className="mt-4">
        <InteractiveLink href="/related-products">
          View Similar Products
        </InteractiveLink>
      </div>
    </div>
  )
}
```

## Styling

The component uses Tailwind CSS for styling and includes:

- Light black background with transparency (black/10)
- Rounded corners (rounded-xl)
- Consistent padding (px-4 py-2)
- Hover state with lighter background (black/5)
- Smooth transition effect
- Centered content using flex layout
- Consistent text styling with Medusa UI Text component

## States

### Default State
- Light black/gray button-like appearance
- Black text
- Rounded corners

### Hover State
- Slightly lighter background
- Smooth transition animation
- Maintains the same size and shape

### Active/Focused State
- Uses default browser focus styling
- Maintains visual appearance

## Accessibility

The component includes several accessibility features:

- Uses proper link semantics with anchor element
- Provides visual feedback on hover
- Supports onClick handlers for enhanced interactivity
- Uses appropriate color contrast
- Maintains consistent text styling

## Examples

### Basic Usage

```tsx copy
<InteractiveLink href="/categories">
  Browse Categories
</InteractiveLink>
```

### With Click Handler

```tsx copy
<InteractiveLink 
  href="/wishlist"
  onClick={() => logAnalyticsEvent('wishlist_viewed')}
>
  View Wishlist
</InteractiveLink>
```

### Within a Navigation Group

```tsx copy
<nav className="flex gap-2 mt-4">
  <InteractiveLink href="/new-arrivals">
    New Arrivals
  </InteractiveLink>
  
  <InteractiveLink href="/bestsellers">
    Bestsellers
  </InteractiveLink>
  
  <InteractiveLink href="/sale">
    Sale Items
  </InteractiveLink>
</nav>
```

### With Custom Props

```tsx copy
<InteractiveLink 
  href="/contact"
  data-testid="contact-link"
  aria-label="Contact customer support"
>
  Contact Us
</InteractiveLink>
```

## Dependencies

- @medusajs/ui (for Text component)
- LocalizedClientLink (for localization support)
- react (for component structure)

## Technical Details

The component:

- Wraps the LocalizedClientLink component for localization support
- Provides consistent styling through Tailwind classes
- Uses the Medusa UI Text component for typography
- Supports all standard link attributes through props spreading
- Maintains a button-like appearance while using semantic link structure
- Has a smooth transition effect on hover for better user experience
- Uses font-normal text weight for better readability
