## CartTotals Component

The `CartTotals` component is responsible for displaying various price breakdowns of a shopping cart, including subtotal, discounts, and total cost.

## Overview

```typescript copy
"use client"

import { convertToLocale } from "@lib/util/money"
import React from "react"
```

### Props

#### `totals` (object, required)
An object containing the pricing details of the cart. The structure is as follows:

| Property         | Type    | Description                                       |
|-----------------|---------|---------------------------------------------------|
| `total`         | number  | The final total amount of the cart.              |
| `subtotal`      | number  | The subtotal before discounts, taxes, and shipping. |
| `tax_total`     | number  | The total tax amount.                            |
| `shipping_total`| number  | The total shipping cost.                         |
| `discount_total`| number  | The total discount applied.                      |
| `gift_card_total` | number  | The total amount covered by gift cards.        |
| `currency_code` | string  | The currency code (e.g., `USD`, `EUR`).          |
| `shipping_subtotal` | number | The subtotal of shipping costs.            |

## Implementation

```typescript copy
const CartTotals: React.FC<CartTotalsProps> = ({ totals }) => {
  const {
    currency_code,
    total,
    subtotal,
    tax_total,
    discount_total,
    gift_card_total,
    shipping_subtotal,
  } = totals

  return (
    <div>
      <div className="flex flex-col gap-y-2 txt-medium text-ui-fg-subtle ">
        <div className="flex items-center justify-between">
          <span className="flex gap-x-1 items-center">
            Subtotal (excl. shipping and taxes)
          </span>
          <span data-testid="cart-subtotal" data-value={subtotal || 0}>
            {convertToLocale({ amount: subtotal ?? 0, currency_code })}
          </span>
        </div>
        {!!discount_total && (
          <div className="flex items-center justify-between">
            <span>Discount</span>
            <span
              className="text-ui-fg-interactive"
              data-testid="cart-discount"
              data-value={discount_total || 0}
            >
              - {convertToLocale({ amount: discount_total ?? 0, currency_code })}
            </span>
          </div>
        )}
        {!!gift_card_total && (
          <div className="flex items-center justify-between">
            <span>Gift card</span>
            <span
              className="text-ui-fg-interactive"
              data-testid="cart-gift-card-amount"
              data-value={gift_card_total || 0}
            >
              - {convertToLocale({ amount: gift_card_total ?? 0, currency_code })}
            </span>
          </div>
        )}
      </div>
      <div className="h-px w-full border-b border-gray-200 my-4" />
      <div className="flex items-center justify-between text-ui-fg-base mb-2 txt-medium ">
        <span>Total</span>
        <span
          className="txt-xlarge-plus"
          data-testid="cart-total"
          data-value={total || 0}
        >
          {convertToLocale({ amount: total ?? 0, currency_code })}
        </span>
      </div>
      <div className="h-px w-full border-b border-gray-200 mt-4" />
    </div>
  )
}

export default CartTotals
```

### Example Usage

```tsx copy
import CartTotals from "@/modules/common/components/cart-totals";

const cartTotalsData = {
  total: 5000,
  subtotal: 4000,
  tax_total: 500,
  shipping_total: 500,
  discount_total: 1000,
  gift_card_total: 500,
  currency_code: "USD",
  shipping_subtotal: 500,
};

export default function CartPage() {
  return <CartTotals totals={cartTotalsData} />;
}
```

## Features
- Displays cart breakdown including subtotal, discounts, and final total.
- Supports currency localization via `convertToLocale` function.
- Conditionally renders discounts and gift card amounts when applicable.
- Provides a clear and structured summary of the cart for the user.
- Easy integration into any checkout or cart summary page.

## Configuration

### Styling Example
If you want to change the text color and spacing, you can modify the component's CSS:

```css copy

  color: #333;
  padding: 1rem;
}

.cart-totals .total {
  font-weight: bold;
  font-size: 1.2rem;
}
```

Or, if using TailwindCSS:

```tsx copy
<div className="p-4 text-gray-800">
  <CartTotals totals={cartTotalsData} />
</div>
```
